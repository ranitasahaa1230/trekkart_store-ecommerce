# Trekkart Store

Trekkart Store is is an e-commerce platform for the ones who love travelling, Get Your Best Deal.

## Contains

- Home Page
- Product Listing Page
- Filters by
Price
Category
Ratings
Search
- Cart Management
- Wishlist Management
- Authentication
Sign-up
Login
Logout
- Single Product Page
- Order Summary
- Loading & Alerts

# Built with -

- React JS
- React Context API + useReducer
- React Router v6
- Vanilla CSS integrated with Kolpotoru UI Component Library
